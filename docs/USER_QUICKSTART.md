# SwiftDrop Quick Start

## Start a Session (Host)
1. Open SwiftDrop.
2. Click **Create Room**.
3. Share room code or QR with guests.

## Join a Session (Guest)
1. Open SwiftDrop.
2. Enter the room code (or scan QR).
3. Join and wait for host approval if required.

## Send a File
1. Select a target peer.
2. Choose file/folder.
3. Confirm send request.
4. Receiver accepts transfer.

## Broadcast to Everyone (Host)
1. Click **Broadcast**.
2. Choose file.
3. Guests receive transfer requests automatically.

## Transfer Status
- Queue and progress appear in the transfer panel.
- Completion is confirmed after receiver-side finalization.

## Theme and Preferences
- Toggle light/dark mode from top bar.
- Theme preference is saved per user/session identity.

## Leave or End Session
- Guests: use **Leave Room**.
- Host: use **End Session** to close the room for all.

## Tips
- Keep devices on same Wi-Fi/LAN.
- Avoid switching networks during transfer.
- For very large files, keep the app foregrounded.
